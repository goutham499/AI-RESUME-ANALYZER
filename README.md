# AI Resume Analyzer

## Project Description
AI Resume Analyzer is a Machine Learning based application that analyzes resumes and predicts the job category based on the skills and experience mentioned in the resume.

The system helps recruiters automatically classify resumes and helps candidates understand which job role their resume best matches.

## Features
- Resume text extraction
- Skill detection
- Job role prediction
- Resume classification using Machine Learning
- Simple user interface

## Technologies Used
- Python
- Natural Language Processing (NLP)
- Machine Learning
- Scikit-learn
- Pandas
- NLTK
- Streamlit / Flask

## Machine Learning Model
The model is trained using NLP techniques like:
- TF-IDF Vectorization
- Logistic Regression / Naive Bayes

## Dataset
The dataset contains resumes categorized into different job roles such as:

- Data Science
- Web Development
- Java Developer
- Python Developer
- DevOps Engineer
